# ReportGenerator

Реализован функцонал, указанный в файле "Report generator.txt", пункты a) - x)

Принятые решения:
  
  - Перенос слов в поле таблицы:
    
    - Если слово не умещается в оставшееся место, 
      но умещается во всю ширину столбца, 
      то слово переносится на новую строку.
    
    - Если слово не умещается в оствашееся место 
      и НЕ умещаестя во всю ширину столбца, слово 
      разбивается посередине, причем перед разбиением 
      строка не переностится, а используется оставшееся 
      место
    
    - Все символы, в том числе пробелы остаются в строке поля.
  
  - Перенос строк по страницам и разделители строк:
    
    - Логика аналогична переносу слов в стобце
    
    - Если строка не умещается в оставшееся место на 
      странице, но умещается на новой странице, 
      то строка переносится на новую страницу.
    
    - Если строка не умещается в оствашееся место 
      и НЕ умещается на новой странице, строка разбивается 
      посередине, причем перед разбиением страница не переностится, 
      а используется оставшееся место
    
    - Разделитель строк, состоящий из символов "-" ставится всегда.

  - Используемые сторонние библиотеки:
    - для чтения и расшифровки файла с данными tsv используется
      библиотека от http://www.univocity.com/


По сборке и запуску.
Задание реализовано с помощью IDE NetBeans 8.0.2 на платформе jdk1.7.0_80. 
Каталог ReportGenerator - кореновой для проекта

Возможно два способа запуска

--Через Ant
Для сборки проекта нужно выполнить команду 
    ant jar
Для выполнения с файлами ввода/вывода по умолчанию
    ant run
Для выполнения со своими файлами
    java -jar dist\ReportGenerator.jar settings.xml source-data.tsv report.txt
где
	settings.xml 	-	файл настроек
	source-data.tsv -	файл исходных данных
	report.txt	-	файл в который будет записан/перезаписан итоговый отчет.

--Через Java
Для сборки проекта нужно выполнить команду
    javac -d classes -cp classes src\com\github\byhalovie\reportgenerator\*.java -extdirs dist\lib
Для выполнения скомпилированных class файлов
    java -classpath classes;dist\lib\univocity-parsers-2.1.2.jar com.github.byhalovie.reportgenerator.Main settings.xml source-data.tsv report.txt
где
	settings.xml 	-	файл настроек
	source-data.tsv -	файл исходных данных
	report.txt	-	файл в который будет записан/перезаписан итоговый отчет.
