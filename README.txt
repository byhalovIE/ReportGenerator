# ReportGenerator
Реализован функцонал, указанный в файле "Report generator.txt", пункты a) - x)
Принятые решения:
  - Перенос слов в поле таблицы:
    - Если слово не умещается в оставшееся место, но умещается во всю ширину столбца, то слово переносится на новую строку.
    - Если слово не умещается в оствашееся место и НЕ умещаестя во всю ширину столбца, слово разбивается посередине, причем перед разбиением строка не переностится, а используется оставшееся место
    - Все символы, в том числе пробелы остаются в строке поля.
  - Перенос строк по страницам и разделители строк:
    - Логика аналогична переносу слов в стобце
    - Если строка не умещается в оставшееся место на странице, но умещается на новой странице, то строка переносится на новую страницу.
    - Если строка не умещается в оствашееся место и НЕ умещается на новой странице, строка разбивается посередине, причем перед разбиением страница не переностится, а используется оставшееся место
    - Разделитель строк, состоящий из символов "-" ставится всегда.
